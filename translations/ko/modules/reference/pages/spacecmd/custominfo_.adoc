[[ref-spacecmd-custominfo]]
= custominfo_

The following spacecmd commands are available for working with custom keys.



== custominfo_createkey

Create a custom key.

[source]
--
사용법: custominfo_createkey [이름] [설명]
--



== custominfo_deletekey

Delete a custom key.

[source]
--
사용법: custominfo_deletekey KEY ...
--



== custominfo_details

Show the details of a custom key.

[source]
--
사용법: custominfo_details KEY ...
--



== custominfo_listkeys

List all custom keys.

[source]
--
사용법: custominfo_listkeys
--



== custominfo_updatekey

Update a custom key.

[source]
--
사용법: custominfo_updatekey [이름] [설명]
--
