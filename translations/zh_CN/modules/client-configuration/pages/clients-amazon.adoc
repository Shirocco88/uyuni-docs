[[clients-amazon]]
= Registering {amazon} Clients

This section contains information about registering traditional and Salt clients running {amazon} operating systems.

ifeval::[{suma-content} == true]
[IMPORTANT]
====
{amazon} repository URLs are available from {scc}, but the packages and metadata are provided by Amazon, not by {suse}.
====
endif::[]


[IMPORTANT]
====
Traditional clients are not available on {amazon}{nbsp}2. {amazon}{nbsp}2 clients are only supported as Salt clients.
====

[WARNING]
====
{amazon} instances always have the same `machine-id` id at `/etc/machine-id`. Make sure you regenerate the `machine-id` after the instance is created. For more information, see xref:administration:tshoot-registerclones.adoc[].
====
