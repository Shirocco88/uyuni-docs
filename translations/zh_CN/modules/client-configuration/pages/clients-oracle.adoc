[[clients-oracle]]
= 注册 {oracle} 客户端

本节包含有关注册运行 {oracle} 操作系统的传统客户端和 Salt 客户端的信息。

ifeval::[{suma-content} == true]
[IMPORTANT]
====
{oracle} 储存库 URL 可从 {scc} 获取，但软件包和元数据由 Oracle 而不是 {suse} 提供。 {oracle} 基础媒体储存库可从 https://yum.oracle.com/ 免费下载。
====
endif::[]


[IMPORTANT]
====
传统客户端不适用于 {oracle}{nbsp}8。 仅当 {oracle}{nbsp}8 客户端为 Salt 客户端时才受支持。
====


== 添加软件频道

将 {oracle} 客户端注册到您的 {productname} 服务器之前，您需要添加所需的软件频道，并同步这些频道。

ifeval::[{suma-content} == true]

====

有关激活密钥的详细信息，请参见 xref:client-configuration: activation-keys.adoc[]。

== 注册客户端

{oracle} 客户端的注册方式与所有其他客户端的注册方式相同。 有关详细信息，请参见 xref:client-configuration:registration-overview.adoc[]。


[WARNING]
====
要注册和使用 {oracle}{nbsp}6 客户端，您需要配置 {productname} 服务器以支持较旧类型的 SSL 加密。 有关如何解决此错误的详细信息，请参见 xref:client-configuration:tshoot-clients.adoc[] 中的``注册较旧的客户端``。
====
