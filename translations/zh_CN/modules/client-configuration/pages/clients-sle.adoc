[[clients-sle]]
= 注册 {sle} 客户端

本节包含有关注册运行以下 {sle} 操作系统的客户端的信息：

* {sles}{nbsp}15{nbsp}SP1
* {sles}{nbsp}15{nbsp}SP2


请按照本章中的说明准备所有 {sle} 产品，包括 {sles} SAP，{sle} Desktop、{sle} HPC 和 {sle} Real Time。 您也可以按照这些说明准备较旧的 {sle} 版本和服务包。

== 注册客户端

要注册您的 {sle} 客户端，需要有引导储存库。 默认会自动创建并且每天会为所有同步的产品重新生成引导储存库。 您可以在命令提示符处使用以下命令手动创建引导储存库：

----
mgr-create-bootstrap-repo
----

有关注册客户端的详细信息，请参见 xref:client- configuration:registration-overview.adoc[]。
